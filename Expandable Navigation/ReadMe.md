# üì± Install the Power Apps Canvas App (.msapp)

This repository contains a Power Apps **Canvas App (.msapp)** file that you can download and import into your own Microsoft Power Apps environment.

Follow the steps below to install the app.

---

## üì• 1. Download the `.msapp` File

1. Go to the `/msapp` folder (or the file shown above).
    
2. Click on the `.msapp` file.
    
3. Select **Download** to save it to your computer.
    

---

## üöÄ 2. Import into Power Apps

1. Open the **Power Apps Maker Portal**  
    [https://make.powerapps.com](https://make.powerapps.com)
    
2. In the left navigation pane, select **Apps**.
    
3. At the top, click **Upload** (or **Import Canvas App**, depending on your interface).
    
4. Choose **Browse** and select the downloaded `.msapp` file.
    
5. Once the upload is complete, Power Apps will open the app in the editor.
    

---

## üõ† 3. Save & Publish the App

1. After the app opens, go to **File &gt; Save**.
    
2. Choose your environment and app name.
    
3. Click **Save**.
    
4. Then click **Publish** to make it available to users.
